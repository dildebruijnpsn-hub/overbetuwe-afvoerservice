# Overbetuwe Afvoerservice PWA

Losse PWA-versie voor Overbetuwe Afvoerservice.

## Belangrijk

- Deze preview gebruikt lokale browseropslag.
- Er is geen koppeling met een bestaand extern databaseproject.
- Er is geen bestaande klantdata meegeleverd.
- Koppel later pas een nieuw, apart Overbetuwe Supabase-project.

## Deploy

Importeer deze map als nieuw Vercel-project of push deze map naar een nieuwe GitHub-repository en importeer die repository in Vercel.
